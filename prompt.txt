BLOCK TYPES REFERENCE:

1. HEADING:
   - Required: kind, text
   - Optional: level (1=large, 2=medium, 3=small), color, align

2. PARAGRAPH:
   - Required: kind, text
   - Optional: size, color, bold, italic, align

3. BULLETS:
   - Required: kind, items (array of strings or objects)
   - Optional: color (applies to all)
   - Item options: text, bold, color, subpoints (array)

4. NUMBERED_LIST:
   - Required: kind, items
   - Optional: start (number), color

5. TABLE:
   - Required: kind, rows (2D array)
   - Optional: header (boolean), header_color, header_text_color, font_size, 
     left, top, width, height, col_widths

6. TEXT_BOX:
   - Required: kind, text
   - Optional: left, top, width, height, font_size, bold, italic, 
     color, bg_color, align

7. IMAGES:
   - Required: kind, layout (row/column/grid), items (array with path)
   - Optional: per image - path (required), caption

COLOR FORMATS:
- Named: "red", "blue", "green", "yellow", "orange", "purple", "pink", "brown", "gray", "black", "white"
- Hex: "#FF0000", "#0000FF"
- RGB Array: [255, 0, 0], [0, 0, 255]
